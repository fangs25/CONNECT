# CONNECT

CONNECT is a PyTorch-based framework for paired single-cell multi-omics integration, directional cross-modal embedding mapping, and full-feature modality prediction. It is designed for paired RNA+ATAC and RNA+ADT data, with an optional single-modality augmentation workflow for leveraging additional unpaired single-omic cells.

## Overview

Single-cell multi-omics technologies provide paired measurements across molecular layers, but such paired profiles remain expensive and difficult to scale. CONNECT uses limited paired data as a reference bridge to infer missing modalities and translate representations for new cells.

CONNECT provides:

- Directional latent mapping without forcing heterogeneous modalities into a generic shared latent space.
- Full-feature cross-modal prediction without highly variable gene or peak pre-selection.
- Support for RNA+ATAC and RNA+ADT paired datasets.
- Optional single-modality augmentation using a three-stage training workflow.

## Installation

Clone the repository and create the conda environment:

```bash
git clone https://github.com/fangs25/CONNECT.git
cd CONNECT

conda env create -f environment.yml
conda activate connect-env
```

Install PyTorch separately according to your operating system, CUDA version, and GPU driver. For example, choose the command generated by the official PyTorch installation selector:

```bash
# Example only. Select the correct command for your CUDA/CPU environment.
pip install torch==2.0.1 --index-url https://download.pytorch.org/whl/cu117
# or
pip install torch==2.5.1 --index-url https://download.pytorch.org/whl/cu121
```

## Quick Start

Standard paired-data training:

Run CONNECT scripts from the repository root so that Python can find the local
`connect` package:

```bash
python run_connect.py --data-dir /path/to/dataset
```

`--data-dir` should be the parent directory containing the paired train/test
files:

```text
/path/to/dataset/
  rna_train.h5ad
  adt_train.h5ad
  rna_test.h5ad
  adt_test.h5ad
```

See `run_connect.py` for a complete standard training procedure.

## Single-Modality Augmentation

CONNECT also supports a three-stage workflow:

1. Standard paired-data training.
2. Single-modality augmentation using additional unpaired cells.
3. Latent-space alignment fine-tuning.

See `run_unimodal_augment.py` for the complete workflow.

## Datasets

Input data should be provided as AnnData objects or `.h5ad` files with cells in rows and features in columns.
Datasets used in CONNECT's experiments can be accessed at [Zenodo](https://doi.org/10.5281/zenodo.20490832)

Typical preprocessing settings are:

- RNA: raw counts, `preprocess=False`.
- ADT: CLR normalization, `preprocess=True`.
- ATAC: TF-IDF normalization, for example `preprocess="Andrew"` and `ceiling=True`.
